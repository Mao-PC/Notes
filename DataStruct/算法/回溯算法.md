# 回溯算法

回溯法又称试探法, 当试探到某一步时, 发现原来的选择达不到目标, 就退回一步重新选择, 这种走不通就退回再走的技术被称为回溯法

## 子集

https://leetcode-cn.com/problems/subsets/

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

```
示例:
输入: nums = [1,2,3]
输出:
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```
